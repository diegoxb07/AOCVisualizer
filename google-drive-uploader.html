🚀 Google Drive File Loader

Load NOAA telemetry & video files directly from Google Drive

Method 1: Shareable Link
How to get a shareable link:
  1. Right-click file in Google Drive → Share
  2. Change to "Anyone with the link"
  3. Copy the link and paste below
Works with: .txt, .nc, .mp4
Method 2: File ID (Direct)
How to get File ID: Extract from URL: https://drive.google.com/file/d/FILE_ID/view
Paste just the ID portion
Method 3: Local Upload
Upload files directly from your computer instead of Google Drive.

💡 Tips & Troubleshooting
✓ Tips:
  • Large files (>100MB) may take longer to load
  • Make sure file sharing is set to "Anyone with the link"
  • Supported formats: .txt (telemetry), .nc (NetCDF), .mp4 (video)
✗ If loading fails:
  • Check file sharing permissions
  • Verify the file ID is correct
  • Try the local upload method instead
  • Check browser console for error details
